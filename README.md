# Stress-Release-Model-and-Extensions
This is the code for the stress release model and its extended model; the generation of the simulation directory is based on the method of Barani (2025), and information can be obtained in the relevant article.  Reference: 
>> Im, K., Elsworth, D., Marone, C., & Leeman, J. (2017). The Impact of Frictional Healing on Stick-Slip Recurrence Interval and Stress Drop: Implications for Earthquake Scaling. Journal of Geophysical Research: Solid Earth, 122(12), 10,102-10,117. https://doi.org/10.1002/2017JB014476  
>> Barani, S., Taroni, M., Zaccagnino, D., Artale Harris, P., & Petrillo, G. (2025). A physics-informed stochastic model for the long-term correlation of earthquakes. Communications Earth & Environment, 6(1), 674. https://doi.org/10.1038/s43247-025-02608-3  
(Note that there must be a missing plotting function or MATLAB optimization function in the code. You can replace or adapt it, but the code is complete)
